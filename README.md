# bot_remna

Telegram-бот для удобного управления подписками и нодами VPN-панели **Remnawave**. Бот спроектирован по методологии 12-Factor App и полностью интегрирован с API Remnawave.

---

## 🌟 Основные возможности

### Для пользователей:
* **📅 Управление подписками**: Просмотр статуса подписки, даты окончания доступа и прямой ссылки для импорта. Поддерживается привязка нескольких подписок к одному Telegram-аккаунту.
* **📥 Пошаговое подключение**: Готовые инструкции и быстрый импорт (deep-links / QR-коды) для популярных клиентов: Happ, V2Box, Streisand, Shadowrocket, Hiddify, Karing, NekoBox и v2rayNG на iOS, Android, Windows, macOS и Linux.
* **🎁 Промокоды**: Быстрое продление подписки путём ввода промокода.
* **📱 [NEW] Управление устройствами (HWID)**: Пользователи могут самостоятельно просматривать список подключённых устройств и удалять старые/неактивные устройства для освобождения лимита слотов.
* **📈 [NEW] Аналитика использования**: Отображение моноширинного текстового эмодзи-графика ежедневного потребления трафика за последнюю неделю.

### Для администраторов (`/admin`):
* **👥 Управление пользователями**: Список всех зарегистрированных пользователей, поиск по имени/username, просмотр детальной карточки с лимитами HWID, трафиком и датой окончания подписки.
* **🔑 Управление токенами доступа**: Выпуск одноразовых инвайт-токенов (`/issue_token`) для регистрации новых пользователей с автоматическим заведением аккаунта на панели Remnawave.
* **🎁 Промокоды**: Создание промокодов на бонусные дни с ограничением максимального числа использований.
* **🌐 Управление нодами**: Мониторинг статуса нод (Online/Offline/Disabled), просмотр нагрузки, перезапуск нод и сброс трафика прямо из Telegram.
* **➕ Добавление нод**: Установка и раскатка новых xray-нод на удалённых серверах через Ansible Playbook по SSH прямо из интерфейса бота.
* **📢 [NEW] Массовая рассылка (Broadcast)**: Отправка сообщений (с поддержкой HTML-форматирования, медиа, фото) всем пользователям из базы данных с автоматическим контролем лимитов скорости (Rate limits).

## 🚀 Быстрый запуск

### 1. Настройка окружения
Создайте файл `.env` в корневом каталоге по примеру `.env.example`:
```env
BOT_TOKEN=your_telegram_bot_token
REMNAWAVE_URL=https://your-remnawave-panel.com
REMNAWAVE_TOKEN=your_remnawave_api_token
SUB_DOMAIN=https://your-subscription-domain.com
ADMIN_TG_IDS=12345678,98765432
```

### 2. Запуск локально
Установите зависимости и запустите бота:
```bash
pip install -r requirements.txt
python bot.py
```

### 3. Запуск в Docker
Сборка и запуск контейнера:
```bash
docker-compose up -d --build
```
